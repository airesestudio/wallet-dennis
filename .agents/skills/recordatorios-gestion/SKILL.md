## Regla
1. Cuando el usuario pide recordar algo, se anota como pendiente
2. Al saludar al día siguiente, se recuerda automáticamente
3. Si confirma que ya lo hizo → se borra el pendiente
4. Si no lo hizo → se mantiene para el próximo saludo
