## Saludo matutino
Cuando el usuario dice "Hola", "Buen día" o similar:
1. Saludar cordialmente
2. Hacer `git pull` en todos los proyectos activos
3. Preguntar en qué proyecto(s) va a trabajar hoy
4. Arrancar servidores con `npm run dev` por cada proyecto activo
5. Verificar VM en GCP, opencode web, auto-save daemon

## Revisión semanal (lunes)
- Git status de TODOS los proyectos
- Verificar backup programado
- Verificar scaffold sincronizado
- Token GHP expuesto (recordatorio)
